# Projeto Didatico de SQLite

Versao mais simples para sala de aula.

## Arquivos

- `app.py`: back-end com Flask e SQLite
- `templates/index.html`: pagina HTML
- `static/style.css`: estilo CSS
- `requirements.txt`: dependencia do Flask

## O que a turma aprende

- classe `BancoLivros`
- `CREATE TABLE`
- `INSERT`
- `SELECT`
- `WHERE`
- `LIKE`
- `ORDER BY`
- `UPDATE`
- `DELETE`
- `COUNT`

## Como executar

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python3 app.py
```

Depois abra:

```text
http://127.0.0.1:5000
```
